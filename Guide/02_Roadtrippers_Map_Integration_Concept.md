
# Roadtrippers-Style Map Integration Concept
(Adapted for TripMate – Sri Lanka Only)

---

## Core Concept

Split-Screen Planning Experience:

-------------------------------------------------
| Sidebar Planning Panel | Interactive Map     |
-------------------------------------------------

Left Side (40% width):
- Itinerary list
- Add activity button
- Filters (Beach / Cultural / Wildlife / Adventure)

Right Side (60% width):
- Interactive Sri Lanka map
- Markers for selected destinations
- Route lines between stops

---

## Map Interaction Behavior

- Clicking itinerary item → highlights map marker
- Clicking map marker → scrolls to itinerary card
- Drag reorder itinerary → route updates visually
- Hover marker → preview card tooltip

---

## Visual Styling

Map Style:
- Desaturated base map
- Teal route lines
- Orange active marker
- Minimal labels

Marker Design:
- Circular 40px
- Icon inside
- Soft drop shadow

---

## Mobile Adaptation

Toggle switch:
- “Map View” / “Plan View”

Stack layout:
- Map on top
- Planner below

---

## UX Goal

The map is not decoration.
It is interactive planning feedback.
