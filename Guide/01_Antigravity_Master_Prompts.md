
# TripMate – Antigravity Master Prompts
(Adventure + Premium Minimal | Sri Lanka Only | Inspired by Roadtrippers)

## Global Design System Prompt
Create a modern adventure-meets-premium design system for a Sri Lanka-only collaborative travel planner called TripMate. 
Balance cinematic exploration visuals with structured SaaS layout. Support light and dark themes.

Include:
- Ocean Teal, Jungle Green, Sunset Orange accents
- Warm white light mode
- Deep navy dark mode (not black)
- Clean typography hierarchy
- Soft rounded corners (20px–24px)
- Framer Motion micro-interactions

---

## Landing Page Prompt
Design a cinematic but structured landing page for TripMate inspired by Roadtrippers layout structure.

Hero Section:
- Full screen Sri Lanka landscape
- Bold headline: “Plan Your Sri Lankan Adventure Together”
- Two CTAs
- Subtle gradient overlay
- Clean spacing

Sections:
- Dynamic destination grid
- How it works (3 steps)
- Minimal footer

Avoid tourism brochure feel.

---

## Dashboard Prompt
Design a modern collaborative trip planning dashboard inspired by SaaS layouts and Roadtrippers map-driven structure.

Include:
- Collapsible sidebar
- Top navbar (search, theme toggle, profile)
- Trip cards with progress bars
- Budget summary
- Activity timeline
- Clean grid spacing

---

## Trip Workspace Prompt
Design an immersive but structured trip workspace.

Header:
- Destination banner
- Budget summary
- Member avatars

Tabs:
- Overview
- Itinerary (accordion day layout)
- Expenses (summary cards + chart)
- Members

Use controlled motion animations.

---

## AI Planner Panel Prompt
Design a slide-in AI planner panel.
Structured interface (not chatbot style).
Input: Natural language.
Output: Itinerary suggestion + Budget estimate.
Modern, subtle, premium.
